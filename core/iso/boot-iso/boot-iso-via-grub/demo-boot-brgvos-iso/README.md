

# demo-boot-brgvos-iso




## Live Account

* Void Linux / [Download](https://voidlinux.org/download/)
* Void Linux / [Installation Guide](https://docs.voidlinux.org/installation/live-images/guide.html)

> You can log into these images as `anon` or `root`, and the password is `voidlinux`.


| Account / Root   | Value        |
| ---------------- | ------------ |
| Username         | `root`       |
| Password         | `voidlinux`  |


| Account / Admin  | Value        |
| ---------------- | ------------ |
| Username         | `anon`       |
| Password         | `voidlinux`  |


> run to change password

``` sh
sudo passwd $(whoami)
```


> run to remove password

``` sh
sudo passwd -d $(whoami)
```
